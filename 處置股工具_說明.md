# 台股處置股當沖篩選工具 — 說明

> 對應腳本：`tw_stock_disposition.py`
> 最後更新：2026-04-06

---

## 什麼是處置股？

「**處置股**」是台灣股市主管機關（TWSE / TPEx）對**交易異常、頻繁出現警示**的股票所採取的限制措施：

- 委託量縮小（一次最多下單 499 張）
- 盤中撮合頻率降低（從逐筆改為集合競價，如每 5 分鐘或每 30 分鐘一次）
- 通常是因為**股價漲跌幅異常、成交量暴增**等警示觸發

這些特性讓處置股在**特定條件下成為當沖機會**，但也伴隨更高的流動性風險。

---

## 這個工具在做什麼？

每天在台股交易時間**每小時自動執行**：

```
從 TWSE 抓取當前所有處置股清單
    ↓ 下載日線 + 60分線技術指標
    ↓ 以「當沖友善」的評分邏輯打分
    ↓ 依綜合分排序
    ↓ 模擬當沖進場（當日出場）
    ↓ 產生 HTML 報告 + 寄 Email
```

同時，報告中也包含**處置股 vs 選股工具的雙策略績效比較**（`stock_reports/compare.html`）。

## 自動執行時間

與選股工具相同：**週一～五 09:00 / 10:00 / 11:00 / 12:00 / 13:00**，排程工作名稱 `TW_Stock_Disposition`。

## 核心策略規則

| 規則 | 說明 |
|------|------|
| 資金分配 | 每次倉位 **15%**，最多同時 **3 檔** |
| 停損 | **-5%** 必砍，無例外 |
| 出場時間 | **不留隔夜**，收盤前一定出場 |
| 禁止行為 | ❌ All in、❌ 攤平、❌ 追新聞 |

---

## 報告欄位說明

| 欄位 | 說明 |
|------|------|
| # | 排名（依綜合分） |
| 代號 | 股票代碼（連結 TradingView） |
| 名稱 | 公司名稱 |
| 處置原因 | 從 TWSE 抓取的處置類別與期間 |
| 股價 | 最新收盤價 |
| 週漲幅 / 月漲幅 | 近期漲跌幅 |
| RSI | 相對強弱指數 |
| KD | K值 / D值 |
| 量比 | 今日量 ÷ 20日均量 |
| 停損風險 | **低** = 近 3 根日低未觸及 -5%；**高** = 已有跌幅紀錄，該場次不模擬進場 |
| 日線（趨勢）| 日K評分 + 訊號 |
| 60分線（時機）| 小時K評分 + 訊號 |
| 綜合分 | 最終評分（0–100） |

### 評分顏色

| 顏色 | 分數 | 意義 |
|------|------|------|
| 紅色 | >= 70 | 強烈信號 |
| 橘色 | 50–69 | 值得關注 |
| 灰色 | < 50 | 僅供參考 |

## 當沖模擬下單說明

| 設定 | 值 |
|------|-----|
| 進場條件 | 綜合分 >= 50 且停損風險不為「高」 |
| 每次最多新增 | 2 檔 |
| 最多同時持倉 | 3 檔 |
| 出場方式 | 觸發停損(-5%) 或 13:00 後以當時收盤價出場 |
| 若有隔夜倉 | 強制結算（理論上不應出現） |

## 與選股工具的差異

| | 選股工具 v2 | 處置股工具 |
|--|------------|-----------|
| 分析標的 | 全市場 1,000+ 檔 | 僅當前處置股（通常 5~20 檔）|
| 操作模式 | 波段（5 交易日）| 當沖（當天出場）|
| 停損設計 | 無（持滿 5 天出場）| -5% 強制停損 |
| 最大倉位 | 無（模擬用）| 3 檔 × 15% |
| 評分重點 | 技術 + 基本面 + 法人 | 量能 + 短期動能 |
| 適合 | 波段投資人 | 當沖 / 短線操作人 |

## 免責聲明

處置股流動性較差、撮合頻率受限，波動風險高於一般股票。本工具僅供技術分析參考，不構成任何投資建議。操作前務必了解處置股的限制措施與相關風險。
