## 判斷邏輯：什麼是「慣性支撐均線」？

### Step 1：掃描近2年每個交易日

對每條均線（5MA / 10MA / 20MA / 60MA / 120MA），逐日檢查是否出現「支撐觸碰」事件：

| 條件 | 說明 |
|------|------|
| **前提**：前一日收盤在均線之上 | 確保是從高點「回落」到均線，而非已跌破後的震盪 |
| **觸碰**：當日低點在均線 ±3% 帶內 | 代表股價下探到均線附近 |
| **未跌穿**：當日收盤不低於均線 3% 以下 | 排除跌破太深的情況 |

### Step 2：計算反彈成功率

若觸碰後 **5個交易日內**，任一日收盤漲幅 ≥ 2%，視為「止跌反彈成功」。

```
成功率 = 反彈次數 / 觸碰次數 × 100%
```

### Step 3：選出慣性支撐均線

- 反彈次數**最多**的均線 → 慣性支撐均線
- 若所有均線的反彈次數完全相同 → 標示「暫無符合之均線」
- 若觸碰次數少於2次 → 樣本不足，不納入統計

## 如何解讀與運用

### 情境：持股跌破5MA被賣出，三週後報告顯示「20MA 成功率 78%」

1. **等待時機**：觀察股價是否回測到 20MA 附近
2. **確認觸碰**：當日低點進入 20MA ±3% 帶
3. **量能確認**：成交量縮量（顯示賣壓減輕）
4. **考慮再進場**：收盤回站 20MA 上方時，可考慮分批布局

> 支撐均線是「機率統計」，不是保證。建議配合其他技術訊號（KD、MACD）一起判斷。
