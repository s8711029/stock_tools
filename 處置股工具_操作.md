## 四、如何解讀報告

### Step 1：確認今日處置股有哪些

報告最上方會列出 TWSE 今日公告的所有處置股及處置原因，先了解是哪一類型的處置。

### Step 2：看綜合分排名

- 紅色（>= 70）：量能與動能俱佳，當日進場機會最大
- 橘色（50–69）：有訊號但確信度較低
- 灰色（< 50）：本日不建議操作

### Step 3：優先看量比欄

**量比是處置股最重要的指標。**

| 量比 | 意義 |
|------|------|
| >= 3x | 爆量，資金大量湧入 |
| 2x ~ 3x | 明顯放量 |
| 1.5x ~ 2x | 溫和放量 |
| < 0.5x | 縮量，不建議操作 |

### Step 4：看停損風險欄

- `低`：近 3 根日低沒有觸碰到 -5% 的位置，風險相對可控
- `高`：近期已有較大下跌紀錄，模擬系統**不會**對此進場，建議謹慎

### Step 5：確認訊號與進場時機

查看「日線」欄的訊號說明，確認是否有 KD 黃金交叉、MACD 翻正等多重確認；
查看「60分線」欄確認短線進場時機。

### Step 6：看模擬當沖績效

報告底部的「模擬當沖紀錄」顯示：
- 今日持倉（若現在是盤中）
- 今日已出場（含停損 or 收盤出場）
- 歷史勝率與平均報酬

---

## 五、實際操作建議

```
09:00 收到報告
  → 看綜合分 >= 50 且量比 >= 2x 的股票
  → 量比是當天動態數字，盤中量比比早盤更準確
  → 10:00 / 11:00 再次收報告確認量比是否持續放大

進場後：
  → 設定停損在 -5%（硬規則）
  → 不管盈虧，13:00 前一定要出場
  → 不攤平（跌了就認停損）
```

## 六、雙策略比較報告（compare.html）

`stock_reports/compare.html` 同時顯示：

- 選股工具 v2 的模擬績效（5 天波段）
- 處置股工具的模擬當沖績效

方便觀察兩種策略在不同市場環境下的表現差異。

## 七、常見問題

| 問題 | 解法 |
|------|------|
| 今日處置股清單為空 | 當日 TWSE 無處置股公告，屬正常 |
| 處置股清單只有固定 3 檔（測試用）| TWSE / TPEx API 暫時無法連線，待網路恢復後自動修正 |
| Email 沒收到 | 確認 `stock_email_config.json` 中的 Gmail 應用程式密碼是否有效 |
| 排程沒執行 | 打開工作排程器確認 `TW_Stock_Disposition` 狀態為 Ready |
