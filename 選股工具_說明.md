# 台股選股工具 v2 — 說明

> 對應腳本：`tw_stock_screener_v2.py`
> 最後更新：2026-04-06

---

## 這個工具在做什麼？

每天在台股交易時間（週一～五 09:00～13:00）**每小時自動掃描全市場 1,000+ 支上市股票**，
從技術面、基本面、法人三個角度打分，篩出最值得關注的前 40 名，並寄送 Email 報告。

```
全市場 1,000+ 檔
    ↓ 第一階段：日線掃描 + 成交量過濾（< 1000張排除）
篩選出日線分 >= 30 的候選股（通常 100~200 檔）
    ↓ 第二階段：月營收基本面加分（FinMind）
    ↓ 第三階段：三大法人買賣超（TWSE T86）
    ↓ 第四階段：60分線精算，合成綜合分
依綜合分排名，取前 40 名
    ↓ 產生 HTML 報告 + 模擬下單記錄 + 寄送 Email
```

## 自動執行時間

| 時段 | 執行內容 |
|------|---------|
| 每天 09:00 | 第1次分析 + 09:00 模擬進場 |
| 每天 10:00 | 第2次分析 + 10:00 模擬進場 |
| 每天 11:00 | 第3次分析 + 11:00 模擬進場 |
| 每天 12:00 | 第4次分析 + 12:00 模擬進場 |
| 每天 13:00 | 第5次分析 + 13:00 模擬進場 + **買入提醒偵測** |

> 週末與國定假日自動跳過，不執行、不寄信。

## 報告欄位說明

| 欄位 | 說明 |
|------|------|
| # | 排名（依綜合分） |
| 代號 | 股票代碼，點擊開 TradingView K 線 |
| 名稱 / 類股 | 公司名稱與產業分類 |
| 股價 | 最新收盤價（元） |
| 今日 / 週漲幅 / 月漲幅 | 近期漲跌幅，紅=漲，綠=跌 |
| RSI | 相對強弱指數 |
| KD | K值 / D值 |
| 月營收YoY | 近3個月平均年增率，紅=成長，綠=衰退 |
| 今日量(張) | 今日成交量 |
| 外資(張) | 外資今日買賣超，`**` = 連續 >= 3 天買超 |
| 投信(張) | 投信今日買賣超，同上 |
| 日線（波段）| 日K評分 + 觸發訊號 |
| 60分線（短線）| 小時K評分 + 觸發訊號 |
| 綜合分 | 最終加權評分（0–100），越高越值得關注 |

### 評分顏色說明

| 顏色 | 分數 | 意義 |
|------|------|------|
| 綠色 | >= 60 | 強烈關注 |
| 橘色 | 40–59 | 值得觀察 |
| 灰色 | < 40 | 僅供參考 |
| 黃底列 | 日線 >= 40 且 60分線 >= 40 | 波段 + 短線共振 |

### 變化箭頭

- **▲ 紅色**：比上次執行分數上升 > 3 分（訊號增強）
- **▼ 綠色**：比上次執行分數下降 > 3 分（訊號減弱）

## 模擬下單說明

系統在每個時段（09:00 / 10:00 / 11:00 / 12:00 / 13:00）**各自模擬進場**，5 個時段獨立追蹤。

| 設定 | 值 |
|------|-----|
| 進場條件 | 綜合分 >= 60 |
| 每時段最多新增 | 3 檔 |
| 最多同時持倉 | 10 檔 |
| 持有期間 | 5 個交易日後自動出場 |

---

## 13:00 買入提醒

同時滿足以下三條件才觸發：

1. 今日成交量 >= 前一交易日 **2 倍**（爆量）
2. 今日收盤 > 今日開盤（**紅K**）
3. 綜合分 >= 50

觸發時，Email 頂端會出現橘色警示框，列出符合股票。
