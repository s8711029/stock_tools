# 台股選股工具 v2 — 說明

> 對應腳本：`tw_stock_screener_v2.py`
> 最後更新：2026-04-22

---

## 這個工具在做什麼？

每天在台股交易時間（週一～五 09:00～13:00）**每小時自動掃描全市場 2,000+ 支上市+上櫃股票**，
從技術面、基本面、法人三個角度打分，篩出最值得關注的**上市前 10 名 + 上櫃前 10 名**，並寄送 Email 報告。

```
全市場 2,000+ 檔（上市 + 上櫃）
    ↓ 第一階段：日線掃描 + 成交量過濾（< 1000張排除）
篩選出日線分 >= 30 的候選股（通常 200~400 檔）
    ↓ 第二階段：月營收基本面加分（FinMind）
    ↓ 第三階段：三大法人買賣超（上市 TWSE T86 + 上櫃 TPEX）
    ↓ 第四階段：60分線精算，合成綜合分
依綜合分排名，上市取前 10 名、上櫃取前 10 名（各別獨立）
    ↓ 產生 HTML 報告 + 模擬下單記錄 + 寄送 Email
```

## 自動執行時間

| 時段 | 執行內容 |
|------|---------|
| 每天 09:05 | 第1次分析 + 09:05 模擬進場 |
| 每天 10:00 | 第2次分析 + 10:00 模擬進場 |
| 每天 11:00 | 第3次分析 + 11:00 模擬進場 |
| 每天 12:00 | 第4次分析 + 12:00 模擬進場 |
| 每天 13:20 | 第5次分析 + 13:20 模擬進場 + **買入提醒偵測** + **跌破5MA賣出判斷** |
| 每天 14:00 | **選股推薦 Email + Telegram 寄出**（收盤後完整報告） |

> 週末與國定假日自動跳過，不執行、不寄信。

## 報告欄位說明

| 欄位 | 說明 |
|------|------|
| # | 排名（各市場分別從 1 排起） |
| 代號 | 股票代碼，點擊開 TradingView K 線（上市連結 TWSE、上櫃連結 TPEX） |
| 市場標籤 | 代號旁深藍色 = **上市**；深黃色 = **上櫃** |
| 名稱 / 類股 | 公司名稱與產業分類 |
| 股價 | 最新收盤價（元） |
| 今日 / 週漲幅 / 月漲幅 | 近期漲跌幅，紅=漲，綠=跌 |
| RSI | 相對強弱指數 |
| KD | K值 / D值 |
| 月營收YoY | 近3個月平均年增率，紅=成長，綠=衰退 |
| 今日量(張) | 今日成交量 |
| 外資(張) | 外資今日買賣超，`**` = 連續 >= 3 天買超 |
| 投信(張) | 投信今日買賣超，同上 |
| 日線（波段）| 日K評分 + 觸發訊號 |
| 60分線（短線）| 小時K評分 + 觸發訊號 |
| 綜合分 | 最終加權評分（0–100），越高越值得關注 |
| ⚑整理 | 偵測「量縮整理支撐」型態（大量紅K後縮量整理，現價未跌破支撐）。橘色顯示支撐價與整理天數；無型態顯示 `—` |

### ★ 可進場 標記（綜合分欄位下方）

| 標記 | 條件 |
|------|------|
| 綠底 + `★ 可進場` | 綜合分 60~85 **且** 量增倍數 >= 2x |
| 橘色「分數偏高(>85)」 | 綜合分超過上限 |
| 灰色「量增不足」 | 量能不足 2x |

> ⚑整理 與 ★可進場 同時出現 = 最優先進場信號

### 評分顏色說明

| 顏色 | 分數 | 意義 |
|------|------|------|
| 綠色 | >= 60 | 強烈關注 |
| 橘色 | 40–59 | 值得觀察 |
| 灰色 | < 40 | 僅供參考 |
| 黃底列 | 日線 >= 40 且 60分線 >= 40 | 波段 + 短線共振 |

### 變化箭頭

- **▲ 紅色**：比上次執行分數上升 > 3 分（訊號增強）
- **▼ 綠色**：比上次執行分數下降 > 3 分（訊號減弱）

## 模擬下單說明

系統在每個時段（09:05 / 10:00 / 11:00 / 12:00 / 13:20）**各自模擬進場**，5 個時段獨立追蹤。

| 設定 | 值 |
|------|-----|
| 進場條件 | 綜合分 60~85 且量增 >= 2x |
| 每時段最多新增 | **上市 3 檔 + 上櫃 3 檔（各別獨立計算，每時段最多 6 檔）** |
| 最多同時持倉 | 10 檔 |
| 出場條件 | **13:20 時段，當日現價跌破5日均線自動賣出** |

> 賣出時系統自動寄出「跌破5日均線賣出通知」Email，內含買入日期、成本、今日5MA、報酬率等資訊。

## Telegram 通知

每次報告產生後，自動傳送兩則訊息：

| 訊息 | 內容 |
|------|------|
| 📊 摘要 | 上市+上櫃綜合排名前 3 名（股號、名稱、分數、股價；有 ★可進場 則標示） |
| 🔔 重點注意 | 前10名（上市前10+上櫃前10）中，有 `★可進場` 或 `⚑支撐整理` 的股票；無則省略第二則 |

> Telegram 的篩選範圍與 Email 表格一致（各前 10 名），資料不會出現落差。

---

## 13:20 買入提醒

同時滿足以下三條件才觸發：

1. 今日成交量 >= 前一交易日 **2 倍**（爆量）
2. 今日收盤 > 今日開盤（**紅K**）
3. 綜合分 >= 50

觸發時，14:00 寄出的 Email 頂端會出現橘色警示框，列出符合股票（上市+上櫃皆適用）。

## 跌破5日均線賣出通知

13:20 時段自動判斷所有持倉是否跌破5日均線：

- **判斷邏輯**：當日現價 < 5日均線 → 以現價立即出場
- **通知時機**：13:20 執行後立即單獨寄送（與推薦 Email 分開）
- **通知內容**：股票名稱、代號、買入日期/價格、今日現價、5MA 價位、報酬率、賣出時間
