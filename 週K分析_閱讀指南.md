## 一、報告結構（由上而下）

### ① 統計摘要列 — 快速掌握今日市場強度

| 指標 | 意義 |
|------|------|
| R:R ≥ 1.5 檔數 | 越多代表市場機會越好 |
| 週線多頭檔數 | 趨勢向上的股票數量 |
| 有效時段數 | 今天幾個時段的選股有進來（最多 5 個） |

### ② 各時段摘要表 — 橫向比對訊號強度

- 同一股出現在 09:00 + 13:00 = 訊號持續整天，可信度高
- 只在單一時段出現 = 謹慎觀察

### ③ 持倉中（紫色底）— 確認現有持倉的進出場參考

- 損益% **紅色** = 獲利中，對照「出場目標」判斷是否到位
- 損益% **綠色** = 虧損中，對照「停損」確認防線是否仍在

### ④ 當日新選股 — 依「多時段出現 → R:R 高」排序

越前面的標的越值得優先研究。

## 二、欄位速查

| 欄位 | 怎麼看 |
|------|--------|
| **趨勢** | 週線多頭 = 中長線向上；拉回修正 = 短期回檔但未壞；週線空頭 = 避開 |
| **策略** | 支撐買進 / 低位進場 / 均線支撐 = 有買點；現價觀察 = 暫不追 |
| **R:R** | ≥ 2.0 優先；1.2 ~ 1.9 可考慮；< 1.0 或負數 = 不值得進 |
| **來源標籤** | `09:00 + 13:00` 多時段 > 單一時段；`持倉中` = 已在持倉確認 |
| **法人** | ▲外 + ▲投 = 法人齊買，加分；▼外▼投 = 法人賣出，需謹慎 |
| **進場價** | 建議進場參考價位（≈ 現價 × 1.005 ~ 1.01） |
| **出場目標** | 最近壓力 × 0.99，可設為停利參考 |
| **停損** | 關鍵支撐 × 0.97，跌破即出場 |

## 三、五步驟閱讀法

```
Step 1  看統計摘要 → 今天市場機會多不多？
Step 2  看 R:R ≥ 2.0 的標的 → 優先研究名單
Step 3  看來源標籤 → 多時段出現 = 訊號持續
Step 4  看持倉中 → 停損防線是否仍有效？目標是否到位？
Step 5  R:R 為負 → 代表現價已超過壓力，不追高；持倉者考慮輕倉或出場
```

## 四、隔日進場篩選邏輯

### 篩選條件

| 條件 | 說明 |
|------|------|
| R:R ≥ 1.2 | 最低賺賠比門檻 |
| 趨勢 ≠ 週線空頭 | 不逆勢操作 |
| 策略 ∈ 支撐買進 / 低位進場 / 均線支撐 | 有明確買點，非追高 |

### 加分項目

| 項目 | 加分邏輯 |

| 投信買超 | +8 分 |

## 六、常見判斷情境

| 情境 | 建議做法 |
|------|---------|
| R:R ≥ 2.0 + 多時段 + 法人買 | 積極考慮，明日開盤確認方向後進場 |
| R:R ≥ 1.5 + 單時段 + 法人中立 | 觀察盤中走勢，有支撐守住再進 |
| R:R < 1.0 或為負 | 不進場；持倉者評估是否出場 |
| 週線空頭 + 任何 R:R | 避開，不逆勢 |
| 持倉中 + 損益已達目標 | 考慮分批出場或上移停損 |
| 持倉中 + 接近停損 | 確認防線是否仍在，準備執行停損 |
