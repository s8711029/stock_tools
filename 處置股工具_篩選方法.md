# 台股處置股當沖篩選工具 — 篩選方法與評分說明

> 對應腳本：`tw_stock_disposition.py`

---

## 篩選流程

```
從 TWSE 抓取處置股清單（官方 API）
    │ 備用：TPEx 上櫃處置股
    │
    ▼ 對每檔處置股下載：
    │  - 日線 K 線（3個月，判斷趨勢）
    │  - 60分線 K 線（30天，判斷進場時機）
    │
    ▼ 計算日線評分 + 60分線評分
    │
    ▼ 合成綜合分
    │
    ▼ 依綜合分排序，全數顯示（不設筆數上限）
    │
    ▼ 模擬當沖進場（綜合分 >= 50 且停損風險為「低」）
```

---

## 評分設計理念

處置股評分與選股工具**不同**，特別針對當沖特性調整：

| 差異點 | 選股工具（波段）| 處置股（當沖）|
|--------|----------------|---------------|
| 量能權重 | 一般（+10 分）| 最高優先（最高 +25 分）|
| RSI 上限 | 60（日線）| 55，且過熱扣分更重（-20）|
| 布林帶 | 下緣反彈 +10 | 低檔 +12，近上軌 -15（不追高）|
| 月營收 | 有（+最高 20 分）| **無**（處置股不看基本面）|
| 法人資料 | 有 | **無** |
| 均線偏重 | MA5/10/20/60 | MA5/10/20（不用 MA60）|

## 評分加減分明細

### 量能（最重要）

| 條件 | 加/扣分 |
|------|---------|
| 今日量 >= 20日均量 × 3 倍（爆量）| **+25** |
| 今日量 >= 20日均量 × 2 倍 | +15 |
| 今日量 >= 20日均量 × 1.5 倍 | +8 |
| 今日量 < 20日均量 × 0.5 倍（縮量）| -10 |

> 縮量的處置股不建議操作，系統直接扣分。

### KD 指標

| 條件 | 加/扣分 |
|------|---------|
| KD 黃金交叉 **且** K < 30（低檔交叉）| **+25** |
| KD 黃金交叉（一般）| +15 |
| K < 20 且 D < 20（KD 極低）| +10 |

> 低檔 KD 交叉對處置股特別有效，加分更多。

### MACD

| 條件 | 加/扣分 |
|------|---------|
| MACD 柱狀圖從負轉正 | **+20** |
| MACD 柱狀圖持續增強（且為正）| +8 |

### RSI

| 條件 | 加/扣分 |
|------|---------|
| 25 < RSI < 55（合理區間）| +10 |
| RSI < 20（超賣）| +5 |
| RSI >= 70（過熱）| **-20**（比選股工具扣更多）|

> 處置股過熱不進，過熱代表已有人在拉高出貨。

### 均線

| 條件 | 加/扣分 |
|------|---------|
| MA5 > MA10 > MA20（多頭排列）| +10 |
| MA5 > MA10（部分多頭）| +5 |

### 布林通道

| 條件 | 加/扣分 |
|------|---------|
| 現價在布林區間下方 25% 以內（低檔）| **+12** |
| 現價在布林區間上方 85% 以上（近上軌）| -15 |

> `布林位置 = (現價 - 下軌) ÷ (上軌 - 下軌)`

### 其他

| 條件 | 加/扣分 |
|------|---------|
| 今日收盤 > 昨日收盤 | +5 |

---

## 綜合分計算公式

| 時段 | 公式 |
|------|------|
| 盤中（09:00–13:30）| `60分線評分 × 60% + 日線評分 × 40%` |
| 盤後 | `日線評分 × 70% + 60分線評分 × 30%` |
| 無 60 分線資料 | `日線評分` |

> 綜合分上限 100 分，下限 0 分

---

## 停損風險評估

系統自動判斷每檔的停損風險：

- **低**：近 3 根日K的最低價 > 現價 × 95%（即 3 根以內沒有觸及 -5% 的位置）
- **高**：近 3 根日K最低價 <= 現價 × 95%（近期已有較大下跌，停損可能較快觸發）

**停損風險為「高」的股票，模擬系統不會進場。**

---

## 當沖模擬進場規則

| 設定 | 值 | 說明 |
|------|-----|------|
| 進場條件 | 綜合分 >= 50 | 同時停損風險需為「低」|
| 倉位 | 15%（模擬資本）| 每次資金的 15% |
| 最多持倉 | 3 檔 | 同時不超過 3 個部位 |
| 每次最多新增 | 2 檔 | 避免同時間全部集中 |
| 停損 | -5% | 日低觸及進場價 × 95% 時出場 |
| 出場時機 | 停損觸發 or 13:00 後收盤價出場 | 不留隔夜 |

## 績效統計說明

| 指標 | 說明 |
|------|------|
| 勝率 | 報酬率 > 0 的筆數 ÷ 總筆數 |
| 平均報酬 | 所有出場紀錄的平均報酬率 |
| 平均獲利 | 獲利筆的平均報酬率 |
| 平均虧損 | 虧損筆的平均報酬率 |
| 累計報酬 | 各筆報酬率 × 15%（倉位比例）後加總 |
